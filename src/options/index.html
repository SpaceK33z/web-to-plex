Radarr URL:
Example: https://example.com/radarr or http://192.168.1.100:7878 (no trailing slash!)

Radarr API key:
In Radarr, go to Settings, click on tab General, click on "Show advanced" and copy paste the value next to "API Key". Example: aa756d33242f6g8ffbca2b3963586f21

Radarr storage path:
The path where you want to store movies. Example: /storage/Movies/
